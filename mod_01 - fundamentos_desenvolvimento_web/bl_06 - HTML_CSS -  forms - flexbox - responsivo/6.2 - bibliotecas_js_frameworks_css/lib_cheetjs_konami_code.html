Você conhece o Konami Code?

Origem

O código apareceu pela primeira vez no lançamento de 1986 do Gradius para NES ( Nintendo Entertainment System). Criado pelo programador Kazuhisa Hasimoto, foi implementado como uma ferramenta de desenvolvimento, pois Hasimoto não conseguia passar de uma parte do jogo, sendo citado dizendo "Porque eu era o único que iria usá-lo, certifiquei-me de foi fácil de lembrar." Mais tarde, o código foi programado em muitos jogos da Konami e não-Konami. O uso mais popular do código foi no videogame Contra , um shooter de dois jogadores também para o NES. Em Contra , o uso do código concedeu trinta vidas adicionais, apelidando-o de "código de 30 vidas". Existem muitas variações do código, usado em muitos jogos. O código em si é:

(↑) (↑) (↓) (↓) (←) (→) (←) (→) (B) (A) (START)
